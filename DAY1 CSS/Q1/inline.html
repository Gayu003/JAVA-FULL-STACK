Basic formatting using inline CSS

Inline CSS is a method that applies CSS styling directly to HTML elements using the ‘style’ attribute. This approach allows developers to define styles for individual elements, making it an effective tool for applying unique styles to specific HTML elements.The border-inline-style property is an inbuilt property in CSS which is used to set the individual logical block inline-border-style property values in a single place in the style sheet. It sets the inline border-style top(left), and bottom(right) of the defining border element.
